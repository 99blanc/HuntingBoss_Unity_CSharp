 # 헌팅보스(충북게임아카데미)
 ### - 라운드마다 보스레이드를 하며 보스를 잡아 나오는 아이템들로 무기를 강화하는 게임
---
> 최초 수정일 : 20.08.12
>> 수정일 : 22.07.02
>>> 최종 수정일 : 22.12.01
---
#### 개발 언어 : C#(씨샵)
#### 개발 도구 : Unity Engine, VisualStudio 10(유니티 엔진, 비쥬얼 스튜디오 10)
#### 개발 환경 : Windows 10(윈도우 10)
#### 개발 담당 : 김세현
---
#### 게임 구조 :
##### 제한적인 공간의 필드에서 다양한 패턴을 가진 보스를 상대하여,
##### 쓰러트리고 매 라운드 마다 아이템을 보상으로 받아 무기를 강화하는 구조
---
#### 게임 조작 :
* 플레이어
##### 이동 : W, A, S, D
##### 회피 : L-Shift
##### 약공격 : L-Mouse 
##### 강공격 : R-Mouse
##### 회복 물약 사용 : 1
---
#### 게임 장애물 :
* 패턴
##### 설명 : 보스의 일반 공격
##### 난이도 : 쉬운 난이도
* 기믹
##### 설명 : 플레이어에게 해로운 디버프를 주는 보스의 기믹
##### 난이도 : 비교적 쉬운 난이도
* 전멸기
##### 설명 : 일정 패턴으로 작용하며 플레이어가 파훼하지 못할 시 플레이어 즉사
##### 난이도 : 보통 난이도
---
#### 게임 보상 :
##### 무기 강화 : 보스에게서 얻은 아이템으로 무기를 강화 가능
##### 클리어 타임 : 보스 클리어 타임에 따른 보상 증대
